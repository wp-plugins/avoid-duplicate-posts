Avoid all duplicate posts on homepage (applies to all posts, including widgets).

== Description ==

If you have more than one featured post sections on homepage and don’t want to repeat same post in different featured sections then you can achieve that by using this lightweight plugin

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `avoid-duplicate-posts` folder to the to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Plugin will do it’s job without any configuration.

== Frequently Asked Questions ==

= Can’t see plugin options in WordPress Dashboard =

This is the lightweight plugin without any options, plugin will do it’s job without any configuration.
